# 🌐 BuzzIn — MERN Stack Social Media Platform

BuzzIn is a **full-stack social media application** built using the **MERN stack (MongoDB, Express, React, Node.js)**.  
It enables users to connect, share, and interact through posts, stories, and real-time messaging — with secure authentication and scalable backend services.

---

## 🚀 Features

- 📝 **Post Creation** — Create, edit, and share posts with your followers  
- 📸 **Story Sharing** — Share time-limited stories to engage your network  
- 👥 **Follow System** — Follow/unfollow users and explore new profiles  
- 💬 **Chat Messaging** — Real-time 1:1 chat between users  
- 🧩 **Authentication** — Secure login and session management using **Clerk**  
- ⚙️ **Scalable Backend** — Built with Node.js & Express; asynchronous event handling with **Inngest Webhooks**  
- 📈 **Responsive UI** — Modern React frontend for seamless experience across devices  

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-------------|
| Frontend | React.js, Clerk (Auth) |
| Backend | Node.js, Express.js |
| Database | MongoDB, Mongoose |
| Real-time & Async | Inngest Webhooks |
| Notifications | NodeMailer |
| Styling | Tailwind CSS |

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
cd BuzzIn
```

### 2. Install Dependencies
For both the **client** and **server** directories:
